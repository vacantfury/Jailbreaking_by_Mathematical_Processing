# HarmBench Large

**Size:** 250 prompts
**Purpose:** Main Evaluation (Experiments 1-4)
**Description:** The primary dataset for the official paper results, used to validate Hypotheses H1 and H2.
**Origin:** Randomly sampled 250 prompts from the **HarmBench Standard Text Behaviors** dataset (`harmbench_targets_text.json`) using `seed=42`.
