# HarmBench Small

**Size:** 100 prompts
**Purpose:** Pilot Study / Functional Verification
**Description:** A subset of HarmBench (100 samples) used for testing pipeline mechanics and initial method validation.
**Sampling Method:** Randomly sampled 100 prompts from the **HarmBench Standard Text Behaviors** dataset (`harmbench_targets_text.json`), using `random.sample` with `seed=42`.
